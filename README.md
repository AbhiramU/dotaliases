.aliases
========

My .aliases files for:
 - bash
 - git

## Installation

Clone it into `.aliases` folder:
```bash
git clone git@github.com:stocarul/dotaliases.git .aliases
```

Add to `~/.bash_profile`:
```bash
source ~/.aliases/bash_aliases
```

Add to `~/.gitconfig`:
```bash
[include]
    path = ~/.aliases/git_aliases
```
